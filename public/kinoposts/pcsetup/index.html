My PC Setup
What I install and change on a new installation on my PC and laptop.

#Essential Programs
Essential applications that should be on most computers.
  • 7zip | File archive manager
  • Chrome(or any web browser of choice) | Web browser
    • uBlock Origin is also recommended as an extension.

    uBlock Origin | Adblock | My config file
    Additional uBlock Origin filters (uBlock Origin settings > My filters list > copy and paste selection)

    Hide Shorts on YouTube filters
    ! Hide shorts button in sidebar
    www.youtube.com##ytd-guide-entry-renderer:has(yt-formatted-string:has-text(/^Shorts$/i))

    ! Hide shorts section on homepage
    www.youtube.com##ytd-rich-section-renderer:has(#title:has-text(/(^| )Shorts( |$)/i))
    www.youtube.com##ytd-reel-shelf-renderer:has(.ytd-reel-shelf-renderer:has-text(/(^| )Shorts( |$)/i))

    ! Hide short remixes in video descriptions
    www.youtube.com##ytd-reel-shelf-renderer.ytd-structured-description-content-renderer:has-text(/^Shorts remixing this video$/i)

    ! Hide shorts category on homepage and search pages
    www.youtube.com##yt-chip-cloud-chip-renderer:has(yt-formatted-string:has-text(/^Shorts$/i))

  • IrfanView + plugins | Image viewer
  • mpv + config | Video player
    • A config is used to change the UI and add features, linked is my own config I make and maintain
  • Office (easy to obtain !!) | Productivity software
This list may be updated from time to time, please check back occasionally!

#Other Programs
Programs that are more specialised, install if desired.
  • Steam | Gaming store
  • Discord + Vencord | Social
  • ShareX | Screenshots tool
  • Twinkle Tray | Control brightness (and volume) of monitors
  • WizTree | Disk Space Analyser (I use the portable version)

  • VSCode | Code editor
  • Git | Version control system
  • Python | Programming language
  • Java (via Eclipse Temurin) | Programming language
  • NVM For Windows | node.js version management utility

  • VirtualBox | Virtual machine
  • DaVinci Resolve | Video editor
  • qBittorrent | P2P torrent client
  • Blender | 3D CG software

    • See https://kinoposts.web.app/mksocserver/ for more Minecraft info
  • PrismLauncher | Minecraft version manager (I use the portable version)
  • Eclipse Temurin JDK 17 (Java) | Programming language
    • Version 17 of Java is the officially support version for 1.17 and newer
